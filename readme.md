DEVELOPMENT
-----------

[![Greenkeeper badge](https://badges.greenkeeper.io/unlight/nativescript-angular2-blank.svg)](https://greenkeeper.io/)
1. Install NativeScript
2. Run `tns install`
3. Run `sigh -w -vv`
4. Run `tns livesync android --watch`

UNIT TESTS
----------
```
tns test android
```
